Frequently Asked Questions

Is this completely free to embed?
Yes — no account required, no license fee, no attribution required beyond the "Powered by SmartAss Facts" link that appears inside the widget. Paste the code and you're done.
Will it slow down my site?
No. The widget loads lazily inside an iframe. It has zero external dependencies — no Google Fonts, no analytics SDKs, no CDN libraries. The entire widget is under 12 KB.
Can I use this in a classroom or educational resource?
Yes — and we encourage it. All 10 questions reference peer-reviewed research, and each answer links to a full explainer article. Great as a 5-minute lesson opener or homework discussion prompt.
How often are the questions updated?
The current set of 10 questions covers the most well-documented science myths. We add new question sets periodically — the widget will automatically show the updated version.
Can I request custom questions for my topic area?
We're working on topic-specific versions (history myths, biology myths, etc.). If you have a specific use case, reach out via the About page.
